# Gym Management System

### Purpose
This is a Flask-based website created for Lab 10. It includes:
* **Home Page:** Welcome message and highlights.
* **About Us:** Details about the mission and team.
* **Membership:** Showcasing gym plans.

### How to Run Locally
1. Install Flask: `pip install flask`
2. Run the application: `python app.py`
3. Open `http://127.0.0.1:5000` in your browser.
